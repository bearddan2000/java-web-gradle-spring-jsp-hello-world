# java-web-gradle-spring-jsp-hello-world

## Description
A springboot web app with jsp support.

#W Tech stack
- java
  - springboot
  - jsp
- gradle

## Docker stack
- gradle:jdk11

## To run
`sudo ./install.sh -u`
[Available at](http://localhost)

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
